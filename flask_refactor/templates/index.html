Pet API Documentation

Our API provides endpoints for generating sample pet data following the Swagger Petstore schema.

Random Pet Generator

GET /api/pets/random

Returns a randomly generated pet object with realistic data.

Example Response:

{
  "id": 5782,
  "name": "Pet-a7bc93f4",
  "category": {
    "id": 3,
    "name": "Birds"
  },
  "photoUrls": [
    "https://example.com/pet/photos/4872.jpg",
    "https://example.com/pet/photos/1298.jpg"
  ],
  "tags": [
    {
      "id": 7,
      "name": "friendly"
    },
    {
      "id": 15,
      "name": "playful"
    }
  ],
  "status": "available"
}

Batch Pet Generator

GET /api/pets/random/batch/{count}

Returns multiple randomly generated pets in a single request.

Parameters:

  • count: Number of pets to generate (integer, max 50)

Example Request:

/api/pets/random/batch/3

Example Response:

[
  {
    "id": 1234,
    "name": "Pet-a8b7c6d5",
    "category": { "id": 2, "name": "Cats" },
    "photoUrls": ["https://example.com/pet/photos/1234.jpg"],
    "tags": [
      { "id": 5, "name": "friendly" }
    ],
    "status": "available"
  },
  /* Additional pet objects... */
]
